HTML


Utiliza etiquetas para definir elementos como títulos, párrafos, imágenes y enlaces, funcionando como el esqueleto sobre el que se aplican estilos y funcionalidad.


Que es el HTML


Es el lenguaje de marcado estándar para crear la estructura y el contenido de las páginas web.


Historia del HTML


La primera versión del HTML nace hacía 1989 como un subconjunto de SGML y es especificada mediante un documento que se denomina HTML Tags.


HTML5


Es la quinta y última versión del estándar HTML, diseñada para estructurar y presentar contenido web de forma semántica, moderna y multimedia sin necesidad de plugins externos.
